# What is it?
> Krappy is a Discord bot **template** generator. It does ***NOT*** generate Discord bots.

---

# Why the name?
> Krappy is a joke name, mocking [Slappey](https://github.com/stuyy/slappey), the outdated `Discord.JS` template generator.
